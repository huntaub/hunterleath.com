Hackathons

Here is a listing of the couple of Hackathons that I have participated in the past two years and associated projects.

HackUVA - February 28 - March 1, 2015

Bullder- (pronounced bull-der-dash) is a localized file-sharing service in the spirit of Yik Yak. This web application gives users the ability to anonymously post photos, files, comments, and votes within a certain radius of their location. It utilizes WebRTC to enable this services without the use of servers by creating a peer-to-peer localized sharing network.

Code · Built in Javascript

HackNC - March 1-2, 2014

Boulder is an exercise web application and Pebble application that enables automatic tracking of specific activity repetitions by performing categorization on the watch's accelerometer data.

Code · Built in Python (Django) and Javascript

Past Projects

UVa 2016 Class Website (Fall 2013 - Present)

I currently manage the Class of 2016 website for the University of Virginia. Built on Django, the site is intended to inform the class of upcoming council events, include engaging content, and handle a large number of event registrations.

1819 (Fall 2012)

1819 was a file-sharing service that existed in the browser. Built on Node.js and Websockets, the server would only aid in the transferring of files while the actual data was stored on the sharer's computer.
